> WordSearchGame Test Task

## Environment

1. **node.js** `v.12.19+`
1. **npm** `v.6.14+`

## Run Server
```bash
$ cd server
# install dependencies
$ npm install

# run serve
$ npm start
```

## Run Client
```bash
$ cd client
# install dependencies
$ npm install

# run development client app
$ npm run serve
```
